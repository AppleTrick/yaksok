# Constants (`src/constants`)

## Purpose
Store immutable, global static values (Colors, Strings, Configs).

## Guidelines
- This folder is **ONLY** for static values.
- No logic or functions allowed here.

## Examples
- `colors.ts`: UI Color Palette.
- `config.ts`: Environment variables or global settings.
- `enums.ts`: Reusable TypeScript enums.
- `strings.ts`: Static text or localized keys.
