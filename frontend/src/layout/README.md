# Layouts (`src/layout`)

## Purpose
Manage structural components that define the frame of the application (Headers, Footers, Sidebars, Wrappers).

## Interaction with Next.js
- These components are typically used inside Next.js `layout.tsx` files.
- They provide the shell for the pages while `app/layout.tsx` handles the Next.js specific layout logic.
- Maintains UI consistency across different routes.
