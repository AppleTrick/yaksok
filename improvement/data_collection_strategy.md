# 영양제/약병 객체 인식 데이터 수집 전략 가이드

데이터 수집이 막막하게 느껴질 수 있지만, 맨땅에 헤딩할 필요는 없습니다. 다음 3단계 전략을 추천합니다.

## 1. 공개 데이터셋 활용 (가장 빠름)
이미 전 세계의 많은 개발자들이 '약병(Pill Bottle)', '음료 캔(Soda Can)', '약(Medicine)' 사진을 찍어서 라벨링까지 해두었습니다. 이것을 가져다 쓰면 됩니다.

- **추천 사이트**: [Roboflow Universe](https://universe.roboflow.com/)
- **검색 키워드**:
    - `Pill Bottle` (약병)
    - `Medicine Container`
    - `Supplement` (영양제)
    - `Vitamins`
- **방법**:
    1. 검색 후 마음에 드는 데이터셋(이미지 + 라벨) 찾기.
    2. 'Download' 버튼 클릭 후 포뱃을 **YOLOv11** (또는 YOLOv8)로 선택.
    3. 다운로드한 파일을 프로젝트 폴더에 넣고 바로 학습 시작.

## 2. 직접 촬영 (가장 정확함)
우리 서비스 환경(책상 위, 손에 들고 찍기 등)에 딱 맞는 모델을 만들려면 직접 찍은 사진이 필요합니다. **"100장만 있어도"** 효과는 확실합니다.

- **준비물**: 팀원들의 영양제, 약병, 스마트폰.
- **촬영 요령**:
    - **다양한 각도**: 정면, 위에서, 비스듬히.
    - **다양한 배경**: 책상, 식탁, 침대 위.
    - **여러 개 배치**: 약병 한 개만 두고 찍기, 여러 개 섞어서 찍기.
- **팁**: 동영상으로 찍으세요!
    - 약병을 놓고 동영상을 10초 정도 찍은 뒤, 프레임 단위로 캡처하면 순식간에 수십 장의 이미지를 얻을 수 있습니다.

## 3. 데이터 증강 (Data Augmentation)
사진 50장만 찍어도 500장 효과를 낼 수 있습니다.
- **Roboflow**나 **Ultralytics** 학습 설정에서 `Augmentation` 옵션을 켜면 됩니다.
- 사진을 좌우 반전, 살짝 회전, 밝기 조절 등을 자동으로 수행하여 데이터를 뻥튀기해줍니다.

## 🚀 추천 액션 플랜

1.  **오늘**: **Roboflow Universe**에 접속해서 `Pill Bottle` 검색. (분명 쓸만한 데이터가 수천 장 있습니다.)
2.  **내일**: 다운로드한 데이터로 모델 학습(Fine-tuning) 한 번 돌려보기. (코랩 등 이용)
3.  **성능 확인**: '칼'이나 '칫솔' 오인식이 사라지는지 확인.
4.  **보완**: 인식이 잘 안 되는 케이스(한국 약병 등)만 직접 촬영해서 추가 학습.
