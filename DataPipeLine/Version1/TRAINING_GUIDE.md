# 🎯 YOLO11 영양제 모델 학습 상세 가이드

본 문서는 데이터 수집부터 V100 서버 학습까지의 전체 실전 프로세스를 설명합니다.

---

## 🏃 [Step 01] 데이터 수집 및 이미지 가공
영양제 통(bottle)과 박스(box) 이미지를 고해상도로 수집하고 학용 해상도(1280x1280)로 통일합니다.

- **위치**: `01_collection/`
- **실행**: `python main.py`
- **결과**: `data/processed/` 폴더에 분류된 이미지 저장

---

## 🏷️ [Step 02] AI 자동 라벨링 및 검수
노가다(수동 라벨링)를 줄이기 위해 AI(YOLO-World)를 사용하여 박스를 자동으로 그립니다.

### 2.1 자동 라벨링 실행
- **위치**: `02_labeling/`
- **실행**: `python auto_labeler_advanced.py --conf 0.1`
- **특징**: 여러 개의 병이나 박스가 한 사진에 있어도 모두 찾아냅니다.

### 2.2 결과 검수 (매우 중요)
- **실행**: `python label_reviewer.py --mode review`
- **조작**:
    - `D` 키: 라벨이 잘 됨 (넘어가기)
    - `R` 키: 라벨이 틀리거나 누락됨 (재라벨링 폴더로 복사)
    - `Q` 키: 종료

---

## 📦 [Step 03] 학습 데이터셋 구성
라벨링된 파일들을 YOLO 학습에 필요한 `Train(80%)`과 `Val(20%)` 구조로 나눕니다.

- **위치**: `03_training/`
- **실행**: `python prepare_dataset.py`
- **결과**: `datasets/supplements/` 폴더 및 `v100_upload.zip` 생성

---

## 🚀 [Step 04] V100 GPU 서버 학습
로컬 PC는 느리므로 강력한 V100 서버에서 학습을 진행합니다.

1. **파일 전송**: 주피터 서버(`http://...`)에 접속하여 `v100_upload.zip` 업로드
2. **압축 해제**: 주피터 터미널 또는 노트북 셀에서 실행
   ```python
   import zipfile
   with zipfile.ZipFile('v100_upload.zip', 'r') as z:
       z.extractall('./')
   ```
3. **학습 실행**: `v100_train.ipynb` 파일을 열고 모든 셀 실행
4. **결과 회수**: 학습이 완료되면 `runs/train/.../weights/best.pt` 파일을 내 컴퓨터로 다운로드

---

## 💡 라벨링 주의사항 (AI도 실수할 수 있습니다)
- **여러 객체**: 사진에 병이 3개라면 박스도 3개여야 합니다. AI가 1개만 찾았다면 `R`로 표시 후 나중에 `labelImg`로 보충하세요.
- **혼합 객체**: 병과 박스가 같이 있으면 각각의 클래스(`0: bottle`, `1: box`)로 라벨링 되어야 합니다.
- **패딩 무시**: 이미지 상하좌우의 검은색 영역(Letterbox)에는 절대로 박스를 그리지 마세요.

## 🏁 최종 산출물
- **`best.pt`**: 서버 학습의 최종 결과물로, 실제 영양제 인식 서비스에 사용되는 핵심 모델 파일입니다.
